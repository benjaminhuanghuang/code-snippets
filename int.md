


## Min and Max of numbers
- Java
```
int min = Integer.MIN_VALUE;
int max = Integer.MAX_VALUE;
```
- C#
```  
int min = int.MinValue;
int max = int.MaxValue;
```
- Python
```
min = -2^(n-1) 
max = 2^(n-1) - 1.

# python 2

>>> sys.maxint
min =  -sys.maxint - 1
```
- JavaScript
// JavaScript use float numbers
```
var max = Number.MAX_VALUE;
var min = Number.MIN_VALUE;
```
